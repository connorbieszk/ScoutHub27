# ScoutHub27
A Scouting app, developed for the BIOCORE 27 FRC Season

## Information
Currently is setup for the REBUILD 26 FRC Season, for testing/offseason competition purposes.

- Builds to a static github page site
  - Also contains a API for storage access
  - PWA, site is useable without wifi access
 
## Use

`pnpm dev` turns on the vite dev preview for both api and webapp
`pnpm build` builds both api and webapp
`pnpm api` runs a api command defined in apps/api/package.json
`pnpm api` runs a web command defined in apps/web/package.json
